# 你好，好记 👋

一个轻量、快速的 Markdown 编辑器。**直接在这里开始写**，或在顶栏点「打开」加载文件。

## 实时渲染

写 Markdown 时，光标所在行显示源码，其余行自动渲染。比如：

- **加粗**、*斜体*、~~删除线~~、`行内代码`
- [链接](https://example.com) 自动识别
- > 引用块会有左边框

## 三个视图

顶栏切换 **实时 / 源码 / 预览**：

- **实时**：边写边渲染（默认）
- **源码**：纯 Markdown 文本
- **预览**：完整渲染（代码高亮、公式、Mermaid 图）

## 快捷键

| 操作 | 快捷键 |
|---|---|
| 加粗 | Ctrl + B |
| 斜体 | Ctrl + I |
| 行内代码 | Ctrl + E |
| 保存 | Ctrl + S |

## 代码块与图表

支持 ``` 代码块（带语法高亮）、表格、`$数学公式$`、Mermaid 流程图。切到「预览」视图可看完整渲染效果。

> 💡 在实时视图里，**点击**渲染后的代码块 / 表格 / 公式，即可进入编辑。

## 还能做

- 🌙 顶栏切换暗色模式（跟随系统）
- 📑 左侧大纲自动生成，点击跳转
- 🔤 顶栏 A+/A− 调字号
- 📥 拖拽 .md 文件到窗口直接打开
